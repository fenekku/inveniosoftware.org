The Invenio RDM project

The Invenio RDM project is an open source collaboration with two main goals:

Repository Platform

Build a turn-key reserach data management (RDM) repository platform based on Invenio Framework and Zenodo.

Community

Grow a community of research institutions, private companies and individuals to sustain the platform going forward.

The project is scheduled to deliver the first release of Invenio RDM by summer 2020

Features
Top of the class UX

Finally a repository that your users will love! We have thought hard about the user experience for all, no matter if they are end-users, curators, system administrators or developers. Expect more of your repository!

Scalability

Invenio RDM scales with you! Handle 1 or 100 million records, 1 byte or several petabytes. It runs on bare-metal, VMs and container platforms such as Kubernetes and OpenShift.

Institutional integration

Invenio RDM integrates with your insitution. It supports your Single Sign-On (SSO) solution, your storage systems, your public/private clouds, your permissions models as well as your custom fields.

Next Generation Repository

Invenio RDM implements the Next Generation Repository (NGR) vision and integrates with other Open Science infrastructures such ORCID, DataCite & OpenAIRE.

Repository Profiles

Invenio RDM comes with pre-configured repository profiles for institutional repositories (IRs), research data management (RDM) systems as well as domain-specific repositories for health and medical sciences.

Resilient

Architected from ground up to be resilient to changing business requirements. Stable APIs, easy maintenance and upgrade, as well as an extensible platform to ensure a reliable core on which you can build new innovative features and integrations.

What is the difference between Invenio RDM and Zenodo?

Zenodo is a repository service hosted by CERN. Invenio RDM is a repository application that anyone can use to run a service similar to e.g. Zenodo.

What is the difference between Invenio Framework and Invenio RDM?

Invenio Framework is a toolbox (a code library) which you can use build a repository application like Invenio RDM, an integrated library system application like Invenio ILS, or other applications requiring powerful search and document management capabilities.

Can I see InvenioRDM in action?

Not yet, but it will not be far from what Zenodo.org is today.

Is the Invenio RDM project a formal project?

The project is an open collaboration like other open-source community projects. There are no formal legal agreements between the project partners.

How do you ensure the success of the project?

The project is designed so that it tolerates a loss of partners except for the two core partners CERN and Northwestern University. The project is further purposely kept short in time to quickly deliver the first product release that all partners can run in production.

How did the project come about?

Zenodo is open-source licensed and thus several institutions tried to reuse the Zenodo source code. Zenodo was however never meant to be installed elsewhere (it is a service, not an application). Several institutions tried to make the same modifications but had no easy way of sharing their changes. These institutions came together to create an open collaboration.

Can I join the project?

Yes. The Invenio RDM project is an open collaboration where anyone can join. The collaboration is characterized by a common goal as well as open and available information sharing and public discussions.

How do I join the project?

To join the project in the initial phase (until summer 2020), we expect you to commit a minimum of 3 person-months of efforts into the project by Summer 2020. If interested, contact the project manager Lars Holm Nielsen.
